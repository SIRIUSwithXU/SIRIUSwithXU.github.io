# 关于github的同步以及该笔记的一些使用说明

详细说明在：[https://chirpy.cotes.page/](https://chirpy.cotes.page/)

## 与git上的仓库同步笔记

1、在github上创建项目

2、使用git clone https://github.com/xxxxxxx/xxxxx.git 克隆到本地

3、编辑项目【增、删、改】
git status   ## 查看修改的状态
git diff .   ## 查看修改的具体不同

4、git add . （将改动添加到暂存区，.代表全部文件）

5、git commit -m "提交说明"

6、git push origin master  将本地更改推送到远程master分支。
